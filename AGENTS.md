# due-rag 项目约束

## RAG 测试服务器

- RAG 测试服务器固定地址为 `http://10.100.123.84:8020`。
- 本地联调、测试计划和示例命令默认使用上述地址，除非用户明确指定其他环境。
- `RAG_SERVICE_API_KEY` 必须通过环境变量或仓库外的安全配置注入，禁止写入源码、配置模板、测试报告或 Git 历史。
- 执行测试时不得打印、回显或记录完整 API Key。
- `/health`、`/ready`、`/docs` 和 `/openapi.json` 是公开检查端点；`/rag/*` 和 `/admin/*` 必须携带 `X-API-Key`。
