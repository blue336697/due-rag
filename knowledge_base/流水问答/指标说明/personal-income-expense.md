# 个人流水报告 — 收支指标

## 月度收支统计

按月份和支付类型（微信/支付宝/银行等）分别统计：
- 每月每种支付类型的收入总金额和笔数
- 每月每种支付类型的支出总金额和笔数
- 统计时剔除退款交易、红包交易、借款履约收入，只保留有效收入

## 收支统计指标（13项）

按4个时间范围（全部/近3月/近6月/近12月）和每种支付类型分别计算。

### 收入指标（7项）

| 指标 | 说明 |
|------|------|
| 单笔最高收入 | 所有收入记录中单笔金额最大值 |
| 单日最高收入金额 | 按天汇总收入，取最大值 |
| 单日最高收入笔数 | 按天统计收入笔数，取最大值 |
| 整额收入金额 | 金额为整数（小数部分为0）的收入总金额 |
| 整额收入占比 | 整额收入笔数 / 总收入笔数 |
| 万元以上收入笔数 | 单笔≥1万元的收入笔数 |
| 万元以上收入金额 | 单笔≥1万元的收入金额总和 |

### 支出指标（6项）

| 指标 | 说明 |
|------|------|
| 大额支出金额 | 单笔≥大额阈值的支出金额总和 |
| 大额支出笔数 | 单笔≥大额阈值的支出笔数 |
| 大额支出占比 | 大额支出金额 / 总支出金额 |
| 单笔最高支出 | 所有支出记录中单笔金额最大值 |
| 单日最高支出金额 | 按天汇总支出，取最大值 |
| 单日最高支出笔数 | 按天统计支出笔数，取最大值 |

## 收支结构分析

### 收入结构（按优先级分类，每笔只归入一类）

| 类别 | 说明 |
|------|------|
| 经营收入 | 自己做生意赚的钱 |
| 工资收入 | 确定是工资的收入 |
| 疑似工资 | 像工资但不能100%确认的收入 |
| 退款收入 | 退货退回来的钱 |
| 借款收入 | 借来的钱 |
| 红包收入 | 收到的红包 |
| 投资收入 | 投资理财赚的钱 |
| 转账收入 | 别人转给你的钱 |
| 其他收入 | 以上都不匹配的收入 |

### 支出结构（按优先级分类，每笔只归入一类）

| 类别 | 说明 |
|------|------|
| 还款支出 | 还信用卡/贷款 |
| 投资支出 | 买理财/基金/股票 |
| 亲属交易 | 转给家人的钱 |
| 消费支出 | 日常消费（车类/家居/医疗/日常） |
| 转账支出 | 转给别人的钱 |
| 其他支出 | 以上都不匹配的支出 |

## 月均收支趋势

- 每月收入/支出金额和笔数
- 按支付方式（微信/支付宝/银行卡）拆分
- 月均收入 = 总收入 / 覆盖月数
- 月均支出 = 总支出 / 覆盖月数
