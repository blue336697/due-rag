# 对公流水 — 对手方分析（M070-M075）

## 共用前置

按对手方聚合两套字典：
- **流入方**：剔除内部转账/借贷流入/利息后的所有收入聚合，取前10名
- **流出方**：剔除内部转账/偿还借贷后的所有支出聚合，取前10名

## 指标

| 指标 | 名称 | 含义 |
|------|------|------|
| M070 | 前10大流入方（含月度） | 金额前10名流入方：排名、名称、累计金额、占比、笔数、逐月明细 |
| M071 | 对方流入构成 | TOP10流入方各自按cate1拆解，看每个大客户的钱是哪种性质（经营/投资/借贷等） |
| M072 | 对方类型标注 | TOP10流入方各自的对手方类型（cate2）和业务标签（cate3） |
| M073 | 前10大流出方（含月度） | 金额前10名流出方：排名、名称、累计金额、占比、笔数、逐月明细 |
| M074 | 对方流出构成 | TOP10流出方各自按cate1拆解 |
| M075 | 前10大流入流出重合方 | 同时出现在流入和流出的对手方，算出重合金额和重合比例——值越高说明该对手方"既给你钱也收你钱"，可能是循环交易 |
