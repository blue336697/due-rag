# 个人流水报告 — 场景识别

场景识别针对每种支付类型和每种时间范围（全部/近3月/近6月/近12月）分别计算。

## 租赁场景

识别手机、电脑等数码产品租赁行为。

| 指标 | 含义 |
|------|------|
| 租赁机构数 | 涉及多少个租赁机构 |
| 租赁月均履约金额 | 平均每月付多少租金 |
| 租赁期内履约金额 | 租期内总共付了多少租金 |
| 租用平台数 | 实际在多少个平台租了东西 |
| 申请台数 | 总共申请了多少台设备 |
| 取消台数 | 取消了多少台 |
| 通过台数 | 实际租了多少台（申请 - 取消） |
| 租用台数 | 实际租用中（不满12期）的台数 |
| 押金 | 实际支付了多少押金 |
| 已付金额 | 已支付的总金额 |
| 剩余待还 | 还要还多少钱 |

## 先享后付场景

识别花呗、白条、分付等先消费后付款的行为。

| 指标 | 含义 |
|------|------|
| 机构数/平台数 | 涉及多少个先享后付平台 |
| 月均履约金额 | 平均每月要还多少 |
| 期内履约金额 | 总共还了多少 |
| 笔数/金额 | 总共消费了多少笔、多少钱 |

## 借款履约场景

识别借款和还款行为。

| 指标 | 含义 |
|------|------|
| 机构数/平台数 | 涉及多少个借款平台 |
| 月均履约金额 | 平均每月要还多少借款 |
| 期内履约金额 | 总共还了多少借款 |
| 借款笔数/金额 | 总共借了多少笔、多少钱 |
| 还款笔数 | 总共还了多少笔 |
