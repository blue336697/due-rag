# 个人流水报告 — 核心指标

以下指标仅针对银行卡类型数据计算。

| 指标 | 含义 |
|------|------|
| 当前就职公司 | 最近一次工资收入的对手方名称 |
| 银行卡余额 | 最新月份的月末余额 |
| 借款平台数 | 收入方向+信用还款特征的对手方去重数 |
| 借款笔数 | 收入方向+信用还款特征的笔数 |
| 借款金额 | 收入方向+信用还款特征的金额总和 |
| 履约平台数 | 支出方向+信用还款特征的对手方去重数 |
| 履约笔数 | 支出方向+信用还款特征的笔数 |
| 履约金额 | 支出方向+信用还款特征的金额总和 |
| 核验月均收入 | 有效收入总额 / 流水覆盖月数 |
| 收入稳定性评分 | 每月收入的变异系数（标准差/均值×100%），数值越小越稳定 |
| 月均收入 | 有效收入总额 / 流水覆盖月数 |
| 月均支出 | 有效支出总额 / 流水覆盖月数 |
| 信贷还款金额 | 支出方向+信用还款特征的金额总和 |
| 收入还款金额 | 有效收入总金额 |
| 月均收支比 | 月均收入 / 月均支出 × 100%，大于100%说明收大于支 |
| 负债收入比 | 信贷还款金额 / 有效收入总额 × 100%，反映还款负担占收入的比例 |

## 收入稳定性评分

1. 算出每个月的有效收入总额
2. 计算所有月份收入的均值和标准差
3. 稳定性评分 =（标准差 / 均值）× 100%
4. 数值越小说明收入越稳定，0 表示每月收入完全一样

## 对手方分析

按交易对手方、月份、支付类型分别统计每个对手方的：
- 总交易笔数、总交易金额
- 收入金额和笔数、支出金额和笔数

## 大资金流向

按微信、支付宝、银行卡分开，看主要资金来往：
- **左侧（收入方）**：净收钱的对手方 TOP20
- **右侧（支出方）**：净付钱的对手方 TOP20
- 过滤掉收入和支出都小于100元的小额对手方

## 银行卡分析

每张银行卡的：交易笔数、存入金额（收入+提现+退款）、支出金额

## 银行流水汇总

按月度和银行卡类型统计：
- **余额统计**：月末余额、月均余额
- **收入分类**：总收入、借款收入、工资收入、存款收入、其他收入
- **支出分类**：总支出、消费支出、还款支出、转账支出、其他支出
