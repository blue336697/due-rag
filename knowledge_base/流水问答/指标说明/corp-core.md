# 对公流水 — 核心速览（M001-M020 + M091）

## 基本信息

| 指标 | 名称 | 含义 |
|------|------|------|
| M091 | 流水文件概览 | 所有上传流水文件的起止时间、月数等汇总信息 |
| M001 | 分析主体数 | 流水中不同"自己人"账户主体名称去重计数 |
| M002 | 分析账户数 | 按"主体+账号"去重计数 |
| M003 | 数据时长（月） | 覆盖了多少个不同的自然月 |
| M004 | 日均余额（项目级） | 所有账户余额均值的再平均 |
| M005 | 借贷流入合计 | 贷款流入+借款流入的收入金额合计 |
| M006 | 偿还借贷合计 | 偿还贷款+偿还借款的支出金额合计 |
| M007 | 有效流入 | 全部收入排除内部转账/借贷流入/利息后的金额 |
| M008 | 有效流出 | 全部支出排除内部转账/偿还借贷后的金额 |
| M009 | 核心有效流入方 TOP5 | 有效流入口径下金额前5名对手方 |
| M010 | 核心有效流出方 TOP5 | 有效流出口径下金额前5名对手方 |

## AI 决策建议

| 指标 | 名称 | 含义 |
|------|------|------|
| M011 | 决策建议结论【AI】 | 大模型综合全报告给出审批结论：拒绝/关注/准入，含评分和理由 |
| M012 | 风险信号结论【AI】 | 4类风险信号：红线（高风险）、黄线（中风险）、低风险提示、正常项 |
| M013 | 建议核实清单【AI】 | 8-12条核实事项，按风险优先级排序，每条含建议材料 |

## 数据质量

| 指标 | 名称 | 含义 |
|------|------|------|
| M014 | 综合数据质量评级 | 余额、对手方、时间三要素齐全度评分：≥2.5良好、≥1.5一般、否则较低 |
| M015 | 余额不连续/缺失账户数 | 账户内相邻余额记录间隔>7天的账户计数 |
| M016 | 命中数据质量风险账户数 | 质量评分≤2的记录过半的账户计数 |
| M017 | 关联方数 | 命中法人/投资/共同股东关联标签的对手方去重计数 |
| M018 | 需关注对手方数 | 命中业务重叠/大额利息/个人大额/新公司/高风险/涉诉等标签的对手方去重计数 |
| M019 | 疑似关联方数 | 命中疑似关联方标签的对手方去重计数 |
| M020 | 个人敏感支出笔数 | 命中敏感场所关键词或夜间大额（22:00-06:00且金额≥5万）的笔数 |
