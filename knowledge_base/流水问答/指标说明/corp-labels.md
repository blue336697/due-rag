# 对公流水 — 标签打标规则

每条对公流水会被打上 3 个标签维度，后续所有指标计算都依赖这些标签。

## cate1 — 交易性质分类（18种）

按优先级从高到低：专项类（利息/税/人力/水电/租金）→ 筹资类（贷款/借款）→ 经营类（兜底）→ 其他

| 分类 | 含义 | 举例 |
|------|------|------|
| 活期利息 | 银行季度结息 | 3/6/9/12月21日的利息入账 |
| 纳税支出 | 企业纳税 | 增值税、所得税、印花税等 |
| 人力成本 | 员工薪酬 | 工资发放、社保公积金代扣 |
| 水电能源 | 公用事业支出 | 电费、水费、燃气费 |
| 租金支出 | 经营场所租金 | 办公室、厂房、仓库租金 |
| 经营收入 | 主营业务收入 | 销售收入、服务费收入 |
| 经营支出 | 主营业务支出 | 采购支出、服务费支出 |
| 贷款流入 | 银行/机构放款 | 银行贷款到账 |
| 借款流入 | 非银行借款 | 民间借贷、股东借款 |
| 偿还贷款 | 偿还银行贷款 | 银行还本付息 |
| 偿还借款 | 偿还非银行借款 | 民间借贷还款 |
| 投资收益 | 理财/投资回报 | 理财产品赎回、分红 |
| 个人流入 | 个人账户转入 | 实控人/股东个人资金注入 |
| 内部转账 | 同名账户互转 | 同一企业不同账户间调拨 |
| 退款流出 | 退款/退货 | 退货退款、服务费退还 |
| 需关注流入/流出 | 需重点关注的交易 | 异常交易对手方的流水 |
| 其他 | 以上都不匹配的 | 兜底分类 |

## cate2 — 对手方类型（6种）

| 类型 | 含义 |
|------|------|
| 本方 | 本企业的其他账户 |
| 银行 | 银行等金融机构 |
| 非银 | 非银行金融机构（小贷、融资租赁等） |
| 民间 | 个人/民间借贷 |
| 企业 | 一般企业法人 |
| 未知 | 无法判断 |

## cate3 — 业务标签（11种）

| 标签 | 含义 |
|------|------|
| 法人关联 | 交易对手为本企业法人/股东/主要人员 |
| 投资关联 | 交易对手为本企业对外投资/分支机构 |
| 共同股东关联 | 交易对手与本企业有共同股东 |
| 疑似关联方 | 可能为关联方但无法确认 |
| 业务重叠 | 交易对手业务与本企业高度重叠（可能是循环交易） |
| 大额利息 | 利息支出异常大 |
| 个人大额 | 个人对手方大额交易 |
| 新公司 | 成立时间较短的对手方 |
| 高风险 | 命中高风险特征 |
| 异常 | 交易模式异常 |
| 涉诉 | 交易对手涉及诉讼 |

## 打标规则

cate1按18条优先级规则链执行，**首次命中即停止**。这意味着规则间存在语义优先级——专项类优先于筹资类，筹资类优先于经营类，最末为"其他"兜底。cate2和cate3在此基础上独立判定，可多标签叠加。
