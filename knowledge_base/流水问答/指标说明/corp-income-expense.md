# 对公流水 — 收支分析与指标性收支（M049-M069）

## 收支分析（M049-M054）

| 指标 | 名称 | 含义 |
|------|------|------|
| M049 | 账户月度流入 | 按"账户×月份"汇总流入金额 |
| M050 | 账户流入合计与均值 | 每账户的流入合计、活跃月数、月均值（仅活跃月取均值） |
| M051 | 账户月度流出 | 按"账户×月份"汇总流出金额 |
| M052 | 账户流出合计与均值 | 每账户的流出合计、活跃月数、月均值 |
| M053 | 项目月度流入流出净额 | 跨账户按月汇总流入、流出、净额（流入−流出） |
| M054 | 项目月均流入流出净额 | 总流入÷总月份数、总流出÷总月份数、净额 |

## 收支构成（M055-M063）

| 指标 | 名称 | 含义 |
|------|------|------|
| M055 | 流入构成 | 所有收入按cate1分类（经营/借贷/投资/内部转账/其他等），输出各类占比和分布 |
| M056 | 经营流入小计 | 经营收入+个人流入 |
| M057 | 筹资流入小计 | 贷款流入+借款流入 |
| M058 | 投资流入小计 | 活期利息+投资收益 |
| M059 | 其他流入小计 | 内部转账+需关注流入+其他 |
| M060 | 流出构成 | 所有支出按cate1分类（经营/纳税/人力/水电/租金/偿还借贷/内部转账等） |
| M061 | 经营流出小计 | 经营支出+纳税+人力+水电+租金 |
| M062 | 筹资流出小计 | 偿还贷款+偿还借款 |
| M063 | 其他流出小计 | 内部转账+需关注流出+其他+退款流出 |

## 指标性收支（M064-M069）

| 指标 | 名称 | 含义 |
|------|------|------|
| M064 | 水电能源支出 | 水电能源类支出的累计、月均、逐月明细 |
| M065 | 租金支出 | 租金类支出的累计、月均、逐月明细 |
| M066 | 人力成本 | 人力成本支出的累计、月均、逐月明细 |
| M067 | 纳税支出 | 纳税类支出的累计、月均、逐月明细 |
| M068 | 月度现金流还原明细 | 按"账户×月份"逐月还原经营净流入→可支配自由现金流（经营净流入−偿债） |
| M069 | 现金流还原结论【AI】 | 大模型综合月度现金流序列给出评级（优秀/良好/一般/关注/高风险）、趋势判断、关键发现和风险因素 |

## 现金流还原公式链

```
月均经营收入 → 月均经营支出 → 月均经营净流入 → 月均偿债 → 月均可支配自由现金流 → 租金保障倍数
```
