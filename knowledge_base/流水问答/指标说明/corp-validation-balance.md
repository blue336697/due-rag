# 对公流水 — 数据验证与余额分析（M021-M038）

## 数据验证（M021-M030）

| 指标 | 名称 | 含义 |
|------|------|------|
| M021 | 余额不连续处数 | 逐笔校验"前条余额±当前金额=理论余额"，与实余额差>0.01元计1处 |
| M022 | 数据时间范围 | 最早交易日~最晚交易日 |
| M023 | 数据时长（账户级） | 最早日到最晚日总天数÷30，最少1个月 |
| M024 | 数据来源 | 从文件属性获取来源描述，去重拼接 |
| M025 | 活期结息校验-缺失次数 | 对比应有季度结息次数与实际命中次数 |
| M026 | 活期结息校验-差异较大 | 理论季度利息vs实际结息偏差>50%视为差异较大 |
| M027 | 文件属性校验 | 检查各文件的勾稽校验和解析报错 |
| M028 | 数据质量评级 | 综合4类异常命中项数评级：≥3较低、1-2一般、0良好 |
| M029 | 数据质量详情 | 汇总各异常的具体描述 |
| M030 | 未提交账户识别 | 当前规则下固定返回空 |

## 余额分析（M031-M038）

基础概念：按账户构造每日EOD（End-of-Day）余额表——当日有交易取最后一笔余额，无交易日沿用前一日余额。

| 指标 | 名称 | 含义 |
|------|------|------|
| M031 | 账户日均余额 | 每账户EOD求和÷天数，项目级取各账户均值再平均 |
| M032 | 账户月日均余额 | 按"账户×月份"的月日均余额矩阵 |
| M033 | 月日均余额均值（账户级） | 所有账户-月份格子的月日均余额算术平均 |
| M034 | 账户期末余额 | 每账户最后一条交易的余额；项目级取算术平均 |
| M035 | 账户月末余额 | 每账户每月的月末余额矩阵 |
| M036 | 账户月末余额均值 | 所有月末余额格子的算术平均 |
| M037 | 项目期末余额 | 各账户期末余额直接相加 |
| M038 | 项目月末余额均值 | 先每月项目级合计，再跨月算术平均 |
