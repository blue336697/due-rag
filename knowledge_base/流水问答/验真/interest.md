# 结息验真

## 术语表

| 术语 | 说明 |
|------|------|
| 结息 | 银行按照约定周期（月/季/年）计算并记入账户的利息 |
| 结息周期 | 银行通常按季度（3、6、9、12月的21日）为客户结算活期存款利息 |
| 日均余额 | 统计周期内 sum（每日账户余额 × 持续天数）/ 统计天数 |
| 活期利率 | 银行对活期存款账户适用的基准利率，各行标准不同 |
| 偏差率 | 理论结息与实际结息金额的可接受偏差范围，超出则触发预警 |

## 场景判定

| 结息场景 | 条件 | 风险提示 |
|---------|------|---------|
| 流水未含结息日 | 流水区间不含任何季末月（3/6/9/12月的21号） | 流水周期未覆盖结息日，无法验真 |
| 覆盖结息日但无结息记录 | 流水覆盖结息日但无结息记录 | 流水覆盖结息日缺失结息记录 |
| 有结息记录但无完整季度 | 有结息但不存在完整季度（当前季度21日至上季21日） | 无法执行计算验真 |
| 偏差率 | 偏差率 = round(｜理论结息 - 实际结息｜ ÷ 理论结息 × 100%, 2) | ≤5% 通过；5%~15% 预警；>15% 拒绝 |

## 提取结息记录

流水中交易备注/摘要包含"结息"、"利息"关键词（可配置）且交易日期=21日的记录。

## 理论结息计算

**季度理论结息** = round(日均余额 × (年利率/365) × 统计天数, 2)

**日均余额** = Σ(每日余额 × 持续天数) / 统计天数

统计天数 = (每季度21号 - 1) - 上季度21号

### 计算步骤

1. 列出每一行的余额和该余额维持的天数
2. 积数 = 每日余额 × 持续天数（下一笔交易日期 - 本次交易日期）
3. 所有积数求和 / 总天数 = 日均余额

注：每日余额取当天最后一笔交易后的余额；节假日没有交易，沿用前一天余额。

## 评分模型

基础分：100 分，逐项扣分（可配置）

| 扣分项 | 扣分 |
|--------|------|
| 无结息记录 | 每笔 -5 分 |
| 偏差率 5%~15% | -20 |
| 偏差率 >15% | -60 |

| 综合评分 | 判定结论 |
|---------|---------|
| 90~100 | 通过 |
| 70~89 | 预警 |
| 0~69 | 拒绝 |

## 各银行活期利率

当银行未覆盖时，默认按 0.05% 计算。
